# README - Paulo André Monteiro - Análise de Dados T1

## Como executar o projeto

1. Abra a pasta do projeto no VS Code
2. Instale a biblioteca pandas pelo terminal: pip install pandas
3. Certifique-se de que o arquivo `Base Varejo.csv` está na mesma pasta do script
4. Abra o terminal (Ctrl + `) e execute: python miniprojeto_paulo_andre_t1.py
5. Os resultados aparecerão no terminal

## Arquivos do projeto
- `miniprojeto_paulo_andre_t1.py` — script principal com toda a análise
- `Base Varejo.csv` — base de dados utilizada
- `README.md` — documentação completa do projeto
